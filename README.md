# dotfiles
My dotfiles, mainly:
- configs for:
  - zsh
  - nvim (with plugins)
  - x11 and more..
- scripts for:
  - status bar ([dwmblocks](https://github.com/airkn/dwmblocks))
  - some features, example: random wallpaper using feh
  - other scripts taken and edited to suit me from [Luke Smith](https://github.com/LukeSmithxyz)
